# Getting Started with Nuxt 3

Nuxt 3 is a powerful framework for building Vue.js applications. In this post, we'll explore the basics of setting up a new project.

## Installation

To create a new Nuxt project, run:

\`\`\`bash
npx nuxi@latest init my-app
\`\`\`

## Directory Structure

Nuxt 3 uses a file-system based routing, meaning that files in the \`pages\` directory automatically become routes.

## Conclusion

Nuxt 3 makes web development a breeze with its auto-imports and server-side rendering capabilities.
