# E-Commerce Platform

This project is a comprehensive e-commerce solution designed to provide a seamless shopping experience.

## Features

- User authentication
- Product catalog
- Shopping cart
- Checkout process
- Admin dashboard

## Tech Stack

- **Frontend**: Nuxt 3, Tailwind CSS
- **Backend**: Supabase
- **Deployment**: Vercel
